Why These Input Fields?

The selected input fields allow users to define the content being audited, the subject area, the depth of fact-checking required, and whether web verification should be used. These inputs ensure the generated prompt is flexible, reusable, and effective across multiple domains.

Prompt Structure

The generated prompt follows a structured hallucination audit workflow: 1. Content Analysis
2. Claim Extraction
3. Confidence Scoring
4. Hallucination Detection
5. Correction Suggestions
6. Source Recommendations This structure encourages critical evaluation rather than simple rewriting.

Recommended AI Tools

  • ChatGPT – Strong reasoning and analysis.
  • Claude – Excellent for long documents.
  • Gemini – Good web integration.
  • Perplexity – Useful for fact-checking with sources.
