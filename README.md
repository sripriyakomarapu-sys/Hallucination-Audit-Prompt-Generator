# Hallucination Audit Prompt Generator

## Overview

The Hallucination Audit Prompt Generator helps users create high-quality prompts for evaluating AI-generated content. It focuses on detecting hallucinations, assigning confidence levels, correcting inaccuracies, and suggesting reliable sources.

## Features

* Content auditing prompt generation
* Fact-checking level selection
* Web verification option
* Multiple output formats
* Copy Prompt functionality

## Recommended AI Tools

* ChatGPT
* Claude
* Gemini
* Perplexity

## Sample Use Case

Input:
"Humans landed on Mars in 2018."

Output:
The generated prompt instructs the AI to identify the claim as hallucinated, explain why it is inaccurate, and provide the correct information.
